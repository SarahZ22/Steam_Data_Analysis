Steam is a digital distribution service for PC games, both allowing users to purchase games from a storefront, download purchased games from a content delivery network, and match with other users for multiplayer gaming. With over 150 million users across 194 countries, Steam represented a deep dataset of gaming data. This project aims to retrieve and visualize the data available from Steam.

We provided the source data we have analyzed, as well as visualizations examining the location of Steam users (both internationally and domestically) as well as a visualization of game sales vs. the user ratings of games. This information is useful for video game publishers, as the location data can tell show where geographically to target advertising/marketing budgets, as well as the effect of user ratings on sales. The location data is also useful for network engineers with video game developers, letting them know where to build server infrastructure so players around the world experience low-latency multiplayer gaming experiences.
