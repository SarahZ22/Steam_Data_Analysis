Dataset Visualizations


Click here for more information

Various visualizations have been created using our database using Tableau and Pandas. Data was scraped/retrieved from SteamSpy and SteamDB. These visualizations were created to explore our data in a way that makes it easy to understand. These visuals compare owners, metascore, game genres, game tags and prices in various charts and visual formats. We are able to add the interactive visualizations directly to this page from Tableau.
